# KAMBETWINS GALLERY

**KAMBE TWINS**（神戸あい・神戸あお）の公式ポートフォリオ＆ギャラリーサイト。

## 技術スタック

- HTML5
- CSS3（グラスモーフィズム、レスポンシブ対応）
- JavaScript（Vanilla JS）

## 構成

| セクション | 内容 |
|------------|------|
| TOP | ヒーローセクション |
| ABOUT | プロフィール紹介カード |
| GALLERY | イラストギャラリー（ライトボックス付き） |
| LINKS & SHOP | X / DLsite / FANZA同人 / Aipictors / Patreon |

## ローカルでの確認

`index.html` をブラウザで直接開くか、ローカルサーバーで起動してください。

```bash
# Python の場合
python -m http.server 8000

# Node.js の場合
npx serve .
```
